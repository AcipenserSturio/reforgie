# reforgie

WIP tool for building Civ 5 mods out of simple, human-readable descriptions.

## Supported features

* Religions

* City-States

* Civilisations (WIP)

## Requirements

* Python >=3.11

* pdm (`pip install pdm`).

## Installation and usage

`pdm install`

`pdm run build`
